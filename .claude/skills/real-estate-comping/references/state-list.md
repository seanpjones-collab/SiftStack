# Disclosure vs Non-Disclosure States Reference

Quick reference for determining which comping methodology to use based on property location.

## Non-Disclosure States

These states do NOT require public disclosure of real estate sale prices. Use the **Triangulation Method** (non-disclosure framework).

| State | Abbrev | Notes |
|-------|--------|-------|
| Texas | TX | Largest non-disclosure market. High property taxes, foundation issues common, hail zones. CAD is primary data source. |
| Utah | UT | Mountain markets with seasonal variations. Growing tech markets (Salt Lake, Provo). |
| Wyoming | WY | Rural markets, limited comparable sales. Resort areas (Jackson Hole) are exceptions. |
| New Mexico | NM | Mixed urban/rural markets. Verify CAD data carefully. |
| Idaho | ID | Rapidly growing markets (Boise). Verify recent trends carefully. |
| Montana | MT | Rural markets, limited comps. Resort areas have unique dynamics. |
| North Dakota | ND | Oil industry influence on certain markets. Rural with limited comps. |
| Alaska | AK | Unique market conditions. Very limited comparable data in most areas. |
| Kansas | KS | Midwest market dynamics. Mixed urban/rural. |
| Mississippi | MS | Flood zones common. Insurance costs factor into affordability. |
| Louisiana | LA | Flood/hurricane zones. High insurance costs. Unique property types (raised homes). |
| Missouri | MO | Mixed urban/rural. St. Louis and Kansas City have more data than rural areas. |

**Total: 12 states**

## Special Considerations by State

### High-Volume Non-Disclosure Markets

**Texas (TX)**
- Largest non-disclosure state by transaction volume
- County Appraisal District (CAD) is primary data source
- Foundation issues are common - always check
- Hail damage zones affect insurance and garage value
- High property taxes impact affordability calculations

**Utah (UT)**
- Growing tech markets (Salt Lake City, Provo, Lehi)
- Seasonal variations in mountain communities
- Ski resort areas have unique dynamics

### States with Partial Disclosure

Some states have varying levels of disclosure:

| State | Notes |
|-------|-------|
| Indiana | Transfer tax stamps can be used to estimate prices |
| Wyoming | Some counties have more data than others |
| Montana | Resort areas may have more available data |

## Data Sources by State Type

### Non-Disclosure States
1. County Appraisal District (CAD) - Physical property data
2. County Clerk/Recorder - Deed of Trust, loan amounts
3. MLS (via syndication) - List prices, DOM, photos
4. Tax records - Assessed values (sanity check only)

### Disclosure States
1. MLS - Full sale price data
2. County Recorder - Deed transfers, sale prices
3. Zillow/Redfin - Aggregated sale data
4. Public records portals - Varies by county
