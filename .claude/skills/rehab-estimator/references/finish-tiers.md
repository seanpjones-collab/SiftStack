# Finish Tiers — Material Quality Definitions

Four finish tiers define the quality level of materials and installation. The choice of tier should be driven by what the **renovated comps (Bucket B)** in the target market have — not by personal preference.

The cardinal rule: **Match the comps. Don't exceed them.**

---

## Tier 1: Builder Grade
**Use for:** Wholetails, rental prep, very low ARV markets (<$100K)
**Cost range:** $15-25/SF (all-in)
**Target buyer:** Investors, budget retail buyers, Section 8 tenants

| Category | Specification |
|----------|--------------|
| **Flooring** | Basic LVP ($1.50-2.50/SF material) or carpet in bedrooms |
| **Kitchen Cabinets** | Existing cabinets cleaned/painted, or cheapest RTA stock (flat panel) |
| **Countertops** | Laminate or butcher block |
| **Appliances** | Basic white or black, not stainless; used/refurbished acceptable |
| **Bathroom** | Clean existing or basic replacements; fiberglass tub surround |
| **Paint** | 1 neutral color throughout (walls + ceiling); trim gets 1 coat if not terrible |
| **Fixtures** | Chrome, basic builder-pack from big box |
| **Hardware** | Basic chrome knobs, $2-3/each |
| **Lighting** | Flush mount dome lights, nothing decorative |
| **Trim** | Existing if intact; replace only what's damaged |
| **Doors** | Existing unless damaged; hollow core replacement if needed |

**Design palette:**
- Walls: White or off-white (SW Alabaster, BM White Dove)
- Trim: Same as walls or basic white
- Cabinets: Paint existing, any neutral
- No accent walls, no two-tone, keep it simple

---

## Tier 2: Mid Grade
**Use for:** Wholetails in nicer areas, low-end flips, $100K-200K ARV markets
**Cost range:** $25-40/SF (all-in)
**Target buyer:** First-time homebuyers, retail buyers in affordable markets

| Category | Specification |
|----------|--------------|
| **Flooring** | LVP throughout living areas ($2.50-4.00/SF material), carpet in bedrooms OK |
| **Kitchen Cabinets** | Painted existing (spray finish) or stock shaker-style (white/gray) |
| **Countertops** | Butcher block or entry-level granite/quartz |
| **Appliances** | New stainless steel, basic models (Frigidaire, Whirlpool base) |
| **Bathroom** | New vanity + mirror + toilet; clean/reglaze tub or basic surround |
| **Paint** | 2 colors max (walls + trim/doors in semi-gloss white); clean, even finish |
| **Fixtures** | Brushed nickel throughout |
| **Hardware** | Brushed nickel or matte black, $4-7/each |
| **Lighting** | Basic updated fixtures (not dome), 1-2 statement pieces OK |
| **Trim** | Replace damaged, add baseboard if missing; 3.25" MDF minimum |
| **Doors** | Replace if damaged; paint all doors for clean look |

**Design palette:**
- Walls: Agreeable Gray (SW 7029) or similar warm greige
- Trim/Doors: Extra White (SW 7006) semi-gloss
- Cabinets: White or light gray
- One consistent look throughout, no room-by-room changes

---

## Tier 3: Investor-Flip Grade (Most Common for Flips)
**Use for:** Full rehab flips, $150K-400K ARV markets
**Cost range:** $35-60/SF (all-in)
**Target buyer:** Retail homebuyers, young professionals, families

This is the sweet spot for most flips. "HGTV ready" without being luxury.

| Category | Specification |
|----------|--------------|
| **Flooring** | LVP throughout main living (higher-end, $3.50-5.00/SF); carpet OK in bedrooms |
| **Kitchen Cabinets** | New stock or semi-custom shaker (white, gray, or two-tone); soft-close |
| **Countertops** | Quartz (white/gray veined) or granite; waterfall edge on islands is bonus |
| **Backsplash** | Subway tile (white 3x6) or herringbone; goes to ceiling behind range |
| **Appliances** | Stainless steel package, mid-tier (Samsung, LG, Whirlpool Gold) |
| **Bathroom** | New vanity (36"+ main, 30"+ secondary), tile floor, tile tub surround, framed mirror |
| **Paint** | Professional 2-3 color scheme; spray doors for smooth factory finish |
| **Fixtures** | Matte black or brushed gold (trending); coordinated throughout |
| **Hardware** | Matching fixture finish, $5-10/each; bin pulls on kitchen drawers |
| **Lighting** | Recessed lights in kitchen + living; pendant over island; updated fixtures everywhere |
| **Trim** | New baseboard throughout (3.25-5.25"), casing around all doors/windows |
| **Doors** | New solid-core or high-quality hollow-core; all matching style |
| **Extras** | Barn door (1-2), open shelving in kitchen, tile to ceiling in master shower |

**Design palette (Modern Neutral — most universally appealing):**
- Walls: Agreeable Gray (SW 7029) or Repose Gray (SW 7015)
- Trim/Doors: Extra White (SW 7006) semi-gloss
- Ceilings: Ceiling Bright White (SW 7007)
- Cabinets: Pure White (SW 7005) or two-tone (white uppers, gray/navy lowers)
- Accent: Iron Ore (SW 7069) for features/accent walls
- Exterior: Repose Gray (SW 7015) or Dorian Gray (SW 7017)

---

## Tier 4: Retail-Premium Grade
**Use for:** High-end flips, $400K+ ARV markets, luxury neighborhoods
**Cost range:** $60-120+/SF (all-in)
**Target buyer:** Move-up buyers, executives, luxury retail

| Category | Specification |
|----------|--------------|
| **Flooring** | Engineered hardwood or wide-plank LVP throughout; heated tile in bathrooms |
| **Kitchen Cabinets** | Semi-custom to custom; inset doors, specialty finishes, organizational inserts |
| **Countertops** | Premium quartz, natural stone, or porcelain slab; waterfall island standard |
| **Backsplash** | Designer tile (zellige, handmade, large format marble-look) |
| **Appliances** | Premium stainless (KitchenAid, Bosch, Café); panel-ready if high-end |
| **Bathroom** | Freestanding tub, frameless glass shower, large-format tile, designer vanities |
| **Paint** | Designer color scheme, possible accent treatments (board & batten, shiplap) |
| **Fixtures** | Luxury brands (Delta, Moen, Kohler premium lines); coordinated suites |
| **Hardware** | Premium hardware, $10-25/each |
| **Lighting** | Designer fixtures, layered lighting plan, under-cabinet LED |
| **Trim** | 5.25"+ baseboard, built-in millwork, custom closet systems |
| **Doors** | Solid-core throughout, upgraded styles (panel, barn door, glass) |
| **Extras** | Smart home features, wine storage, custom mudroom, outdoor living |

---

## How to Select the Right Tier

```
IF ARV < $100K → Tier 1 (Builder Grade)
IF ARV $100K-$200K → Tier 2 (Mid Grade)
IF ARV $200K-$400K → Tier 3 (Investor-Flip Grade)
IF ARV > $400K → Tier 3 or Tier 4 depending on comp finishes

ALWAYS override with comp evidence:
  IF renovated comps have laminate counters → use laminate regardless of ARV
  IF renovated comps have quartz + tile → match that level
  IF market is saturated with flips → may need to go one tier up to compete
```

The comps tell you what the MARKET rewards. Your finish tier should match — never exceed — what the top-selling renovated comps in that micro-market had.
