# Wholetail vs Full Rehab — Scope Comparison

This reference defines exactly which items are in-scope for each exit strategy. Use this to build the two parallel estimates.

---

## Strategy Definitions

### Full Rehab (Investor Flip)
**Goal:** Produce a retail-ready, Bucket B comparable property that commands top-of-market pricing.
**Timeline:** 4-8 weeks typical
**Budget:** $25-75K+ depending on market and condition
**Exit:** Sell to retail homebuyer via MLS at full ARV
**Contingency:** 10-20%

### Wholetail
**Goal:** Clean, functional, move-in ready but NOT show-ready. Fix what's broken, clean what's dirty, maximize ROI per dollar spent.
**Timeline:** 1-3 weeks typical
**Budget:** $3-15K typical
**Exit:** Sell to another investor, owner-occupant, or list on MLS at a discount to ARV
**Contingency:** 5-10%

## Quick Decision Framework

When building the scope for each line item, ask:

**For Full Rehab:** "Would a retail buyer walking through notice this? Would it affect the sale price?" → If yes, include it.

**For Wholetail:** "Is this broken, unsafe, or so ugly it would scare buyers away?" → If yes, include it. If it's just dated but functional, leave it.

---

## Cost Ratio Benchmark

In a well-scoped deal, wholetail costs should typically be **15-30%** of full rehab costs. If the ratio is higher, the property may not be a good wholetail candidate (too much deferred maintenance). If lower, the property may be better as a wholetail than a full rehab (most issues are cosmetic).

| Wholetail/Rehab Ratio | Interpretation |
|-----------------------|----------------|
| < 15% | Strong wholetail candidate — mostly cosmetic issues |
| 15-30% | Typical — clear ROI difference between strategies |
| 30-50% | Significant deferred maintenance — rehab may be better ROI |
| > 50% | Not a good wholetail candidate — too much must-fix work |
