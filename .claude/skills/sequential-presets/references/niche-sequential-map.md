# Niche Sequential Preset Map (Base Template)

This document provides the standard, recommended preset structure for a **Niche Sequential** marketing workflow. Use this as the starting point for building a user's customized plan.

**Folder Name**: `00. Niche Sequential`

| # | Preset Name | Purpose |
|---|---|---|
| 00 | Needs Skipped | Finds all new, unprocessed records from first-to-market lists that have no phone numbers and need to be skip traced. |
| 01 | Skipped No Numbers | Isolates records that were skip traced but yielded no phone numbers, identifying them for a second skip trace attempt or manual research. |
| 02 | Ready to Call | The primary starting point for the marketing cadence; contains all records with phone numbers that have not yet had a call attempt. |
| 03 | FTM Follow Up 1 | Manages the first follow-up call. Captures records that have been called once. |
| 04 | FTM Follow Up 2 | Manages the second follow-up call. Captures records that have been called twice. |
| 05 | FTM Follow Up 3 | Manages the third and final follow-up call before moving to a new marketing channel. |
| 06 | Needs 1st Mail | Catches records that have completed the initial calling sequence (3+ attempts) without contact and are ready for their first direct mail piece. |
| 07 | Mail Monthly | Manages the long-term nurture sequence for uncontacted records, sending them a mail piece once per month. |
| 08 | Vacant Mailing → DP | Segments records with a vacant mailing address for Deep Prospecting, as mail will not reach them. |
| 09 | Return Mail → DP | Segments records where mail has been returned, indicating a bad address that requires Deep Prospecting to find the owner. |
| 10 | No Response DM → DP | After an extended period of no contact via calls or mail, these records are moved to Deep Prospecting for intensive research. |
| 11 | Not Interested Qrtly | Re-engages with owners who were previously not interested, following up on a quarterly basis to catch changes in their situation. |
